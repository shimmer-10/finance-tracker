# 💰 Finance Tracker

A simple and clean web-based personal finance tracker built using Flask and HTML.  
It helps users track daily expenses, manage categories, and set monthly budgets.

---

## 🚀 Features

- ➕ Add new expenses easily
- 📋 View all expenses in a structured table
- 🗂️ Categorize expenses (Food, Travel, Education, etc.)
- 📅 Track expenses with date
- 💰 Monthly budget management (set & update budget)
- 📊 View total spending and remaining budget
- 🔄 Simple navigation between different webpages

---

## 🛠️ Tech Stack

- Python 🐍
- Flask 🌐
- HTML5
- CSS (basic styling)
- Jinja2 Templates

## ⚙️ How to Run the Project

### 1. Clone the repository
```

git clone https://github.com/your-username/finance-tracker.git

```

### 2. Go to project folder
```

cd finance\_tracker

```

### 3. Install Flask
```

pip install flask

```

### 4. Run the app
```

python app.py

```

### 5. Open in browser
```

http://127.0.0.1:5000/
```

---

## 📌 Pages Overview

- **Dashboard (`/`)**
  - Shows summary of expenses and budget

- **Expenses (`/expense`)**
  - Displays all added expenses in a table

- **Add Expense (`/add`)**
  - Form to add new expense

- **Budget (`/budget`)**
  - Set and update monthly budget
  - View total spent, remaining budget, and track overspending
